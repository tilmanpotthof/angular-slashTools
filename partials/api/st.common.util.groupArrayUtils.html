groupArrayUtils
service in module st.common.util

Description

Service to group objects stored in arrays by property values. For example an array of users can be grouped by their cities, even if the city is stored a the attribute like address.city.

Methods

  • group(array, groupDefinition)

    Groups array elements by attributes specified in the groupDefinition.

                 var users = [
                 {username: "tpotthof", company: "//SEIBERT/MEDIA GmbH"},
                 {username: "mclasen", company: "//SEIBERT/MEDIA GmbH"},
                 {username: "dcrockford", company: "Paypal Inc."}
             ];
             groupArrayUtils.group(users, {by: "company", in: "usersForCompany"});

             // Result
             // [{
             //     company: "//SEIBERT/MEDIA GmbH",
             //     usersForCompany: [
             //         {username: "tpotthof", company: "//SEIBERT/MEDIA GmbH"},
             //         {username: "mclasen", company: "//SEIBERT/MEDIA GmbH"}
             //     ]
             // },{
             //     company: "Paypal Inc.",
             //     usersForCompany: [
             //         {username: "dcrockford", company: "Paypal Inc."}
             //     ]
             // }];
