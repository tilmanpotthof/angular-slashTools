objectUtils
service in module st.common.util

Description

Util service to set and get object properties by their string path.

Methods

  • getProperty(o, propertyPath)

    Returns the property of an object described by the propertyPath or undefined if not found. The path can be nested like address.cityin the example below.

                 objectUtils.getProperty({name:"myName"}, "name"); // "myName"
             objectUtils.getProperty({address:{city:"New York"}}, "address.city"); // "New York"
             objectUtils.getProperty({}, "property"); // undefined
    Parameters
    ParamTypeDetails
    oobject

    Object to get the property from

    propertyPathString

    Path to the property (e.g. "address.city")

    Returns
    *

    Returns the property or undefined.

  • setProperty(o, propertyPath, o, createProperty)

    Sets a property to an object with name of the described propertyPath.

    The path can be nested like address.city in the example below, but the nested objects must exists or the function throws an error. This error can we avoided with the optional createProperty parameter.

                 // changes "myName" to "John"
             objectUtils.setProperty({name:"myName"}, "name", "John");

             // changes "New  York" to "L.A."
             objectUtils.setProperty({address:{city:"New York"}}, "address.city", "L.A.");

             // throws an error
             objectUtils.setProperty({}, "address.city", "New York");

             // creates the nested property {address:{city:"New York"}} on the object
             objectUtils.setProperty({}, "address.city", "New York", true);
    Parameters
    ParamTypeDetails
    oobject

    Object to set the property

    propertyPathString

    Path to the property (e.g. "address.city")

    oobject

    Object to be set as property

    createProperty
    (optional)
    		boolean

    If true a nested property path will be created.

    (default: false)

    Returns
    *

    Returns the property or undefined.
