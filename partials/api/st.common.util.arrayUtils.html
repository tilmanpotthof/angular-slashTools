arrayUtils
service in module st.common.util

Description

Util service to simplify working with arrays.

Methods

  • addOnce(element, array, objectEquality)

    Adds an element to an array, if no identical element is in the array and returns true. Otherwise the element is ignored and the function returns false.

    With the optional parameter objectEquality objects are compared by equality not identity with the angular.equals function. .

  • extractPropertiesFromArray(array, propertyPath)

    Extracts properties from an array of objects. Not existing properties are ignored.

    Uses the objectUtils.getProperty function internally.

                 var users = [
             {
                 username: "tpotthof",
                 address: {
                     city: "Mainz"
                 }
             },
             {
                 username: "pmueller",
                 address: {
                     city: "Frankfurt"
                 }
             },
             {
                 username: "aschmidt",
                 address: {
                     city: "Mainz"
                 }
             },
             {
                 username: "nlarssen",
                 address: null
             }];

             // returns ["tpotthof", "pmueller", "aschmidt", "nlarssen]
             arrayUtils.extractPropertiesFromArray(users, "username");

             // returns ["Mainz", "Frankfurt", "Wiesbaden"]
             arrayUtils.extractPropertiesFromArray(users, "address.city");
    Parameters
    ParamTypeDetails
    arrayArray.<*>

    Array with properties to extract

    propertyPathString

    Path to the property (e.g. "address.city")

  • inArray(element, array, objectEquality)
